Why Some Endpoints Look Raw

Nothing is broken here. These tools simply serve different audiences.

/health
Machine-readable JSON for probes and automation. This page is the human-friendly view.
/metrics
Prometheus requires plain-text exposition format, so the raw endpoint stays text by design.
Prometheus and Alertmanager
Prometheus is a query console and Alertmanager only shows alerts when rules fire. The preloaded Grafana dashboard is the polished front-end.
Loki on port 3100
Loki is a logs API, not a standalone web app. Use Grafana Explore for the actual logs UI.

Operator Notes

Helpful defaults so the local stack behaves like a real environment without unnecessary friction.
